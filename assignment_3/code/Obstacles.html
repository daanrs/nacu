Obstacles in CPM

All adhesion values are set to 0. The main difference between the obstacles and the red cell type is the activity. The obstacles have a maximum activity of 0. Furthermore, setting the lambda value of the perimeter to 1 for the red cell type ensures smooth and constant movement of the moving cell. The same lambda value for the obstacles is set to a much higher value. Lastly, the volume parameters are used to specify the size of the objects and the red cell. The ratio between the perimeter and the volume is important; a different ratio could make the obstacles start to interact with either themselves or the moving cell.
